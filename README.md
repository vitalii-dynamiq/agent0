# Agent-0 | DGE Executive Platform

AI-powered executive decision support platform for the Department of Government Enablement (DGE).

## Repository Structure

```
agent0/
├── agent0-ui/          # Next.js frontend application
├── Agent-0DraftSOW*    # Statement of Work documentation
├── Agent-0_Project*    # Project plan documentation
└── Agent0_AWS*         # AWS Discovery documentation
```

## Quick Start

```bash
# Navigate to the UI project
cd agent0-ui

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:4273](http://localhost:4273) to view the application.

## Deployment

The UI application (`agent0-ui`) is configured for Vercel deployment. When importing to Vercel, set the **Root Directory** to `agent0-ui`.
